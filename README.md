# carpal-docs

Бизнес-модель, юнит-экономика, лог решений по проекту [CarPal](https://car-pal-tau.vercel.app/).

## Структура

| Папка | Что внутри |
|---|---|
| `strategy/` | Монетизация, take rates, ADR-lite по продукту и архитектуре |

## Связанные репозитории

- `carpal-backend` — FastAPI + Postgres
- `car-pal-tau` (Vercel) — публичный фронт
- `carpal-marketing` — пре-лонч маркетинг (живёт в `carpal/marketing/`)
